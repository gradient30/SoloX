# QAS Network Agent

The release APK is generated by `scripts/android_agent/package.ps1`.

SoloX never installs or starts the Agent during capability discovery. Installation
and Android VPN authorization require explicit user actions.

Current bundled artifact:

- APK: `qas-network-agent-0.1.0.apk`
- Package id: `io.solox.networkagent`
- App label: `QAS Network Agent`
- Minimum protocol version: `1`

The Agent is a terminal-side Android weak-network proxy. It uses `VpnService` to
capture only the target App UID, forwards TUN traffic through native
`tun2proxy`, and applies the local SOCKS5 shaper profile sent by SoloX.

The on-device UI is localized for test-department use and has four bottom tabs:

- `总览`: VPN authorization, foreground service status, start/stop/refresh.
- `弱网`: target package and TUN/tun2proxy/SOCKS5 data-plane notes.
- `日志`: local Agent logs with level filters.
- `设置`: version, package id, control socket, background operation notes.
