## 📦 依赖管理

### 版本锁定策略

根据 SoloX 项目开发规范，严格锁定关键依赖版本：

```python
# 核心 Web 框架 (严格锁定)
Flask==2.0.3
Werkzeug==2.0.3
Jinja2==3.0.1

# WebSocket 支持 (兼容版本组合)
Flask-SocketIO==4.3.1
python-socketio==4.6.0
python-engineio==3.13.2

# iOS 设备通信 (锁定稳定版本)
tidevice==0.9.7
```

### 依赖冲突解决

- **版本兼容性**: 确保依赖包版本兼容
- **冲突检测**: 自动检测版本冲突
- **降级策略**: 必要时使用稳定的旧版本
