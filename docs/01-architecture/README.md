## 🏗️ 架构概览

SoloX 采用模块化的分层架构设计：

```
┌─────────────────────────────────────────────────────────┐
│                   表现层 (Presentation)                │
├─────────────────────────────────────────────────────────┤
│    Web UI        │    RESTful API    │    Python SDK    │
└─────────────────────────────────────────────────────────┘
                              │
┌─────────────────────────────────────────────────────────┐
│                   业务层 (Business)                    │
├─────────────────────────────────────────────────────────┤
│  路由控制  │  数据处理  │  性能分析  │  报告生成       │
└─────────────────────────────────────────────────────────┘
                              │
┌─────────────────────────────────────────────────────────┐
│                   服务层 (Service)                     │
├─────────────────────────────────────────────────────────┤
│  APM核心   │  设备管理  │  数据收集  │  监控调度       │
└─────────────────────────────────────────────────────────┘
                              │
┌─────────────────────────────────────────────────────────┐
│                   数据层 (Data)                        │
├─────────────────────────────────────────────────────────┤
│  ADB通信   │  iOS通信   │  文件存储  │  日志管理       │
└─────────────────────────────────────────────────────────┘
```

## 🔧 核心特性

- **🔓 免ROOT/越狱**: 无需设备权限提升，降低使用门槛
- **📊 全面监控**: 支持CPU、内存、网络、FPS、电池等多维度监控
- **📈 实时分析**: 提供实时数据可视化和性能分析
- **🔄 跨平台**: 同时支持Android和iOS平台
- **🐍 易集成**: 提供Python API和RESTful接口
- **⚙️ 可扩展**: 模块化设计，易于功能扩展
